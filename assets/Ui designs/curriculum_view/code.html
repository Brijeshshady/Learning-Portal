info Description

This foundational lesson introduces students to the Python programming environment. They will learn how to write a simple script that outputs text to the console, understand the concept of strings, and interact with basic user input to create dynamic responses.

flag Learning Outcomes

  • Understand the print() function and string formatting.
  • Declare and assign simple variables.

assignment Practical Task

Build a "Greeting Bot" script that asks for the user's name and favorite color, then outputs a personalized message combining both inputs.
